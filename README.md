# Sitio web personal — Aldo Castillo Bernal

Este es mi sitio web personal: una mezcla entre portafolio, biografía profesional y pequeño laboratorio donde practico el código que voy aprendiendo.  
El proyecto está construido con HTML5, CSS3 y una buena dosis de paciencia mientras aprendo desarrollo web sin supervisión adulta.

## 🧩 Contenido del sitio

- **Inicio**  
  Presentación general, mezcla de comunicación audiovisual, investigación y mis aventuras aprendiendo a programar.

- **Biografía**  
  Recorrido profesional y académico, con énfasis en medios públicos, cultura digital y mi investigación doctoral.

- **Portfolio**  
  Selección de proyectos audiovisuales, campañas institucionales y contenidos digitales.  
  Incluye un caso de estudio completo: *Aprendiendo en Lenguas Originarias*.

- **Contacto**  
  Un formulario simple para enviar mensajes.

## 🛠️ Tecnologías utilizadas

- **HTML5 semántico**  
- **CSS3** (sin frameworks, a mano, para aprender de verdad)  
- **Git + GitHub Pages** para despliegue  
- **Tipografías:** DM Serif Display y DM Sans
